# Mini Iframe CRM

Тестовый CRM-проект на Laravel для приёма и обработки обращений клиентов.

Проект состоит из двух основных частей:

- iframe-виджет обратной связи для клиентов
- менеджерская панель для просмотра, фильтрации и обработки тикетов

---

## Описание проекта

Пользователь может отправить обращение через встроенный виджет:

- указать имя
- телефон
- email
- тему обращения
- текст обращения
- прикрепить файлы

После отправки заявка сохраняется как тикет, связывается с клиентом и попадает в CRM.

Менеджер в панели может:

- просматривать список обращений
- фильтровать тикеты по email, телефону, статусу и периоду
- открывать карточку обращения
- скачивать прикреплённые файлы
- переводить тикет в статус обработанного
- смотреть статистику по обращениям за день, неделю и месяц

---

## Основной функционал

### Клиентский виджет

Маршруты:

- `/widget`
- `/feedback-widget`

Возможности:
- отправка обращения через форму
- валидация обязательных полей
- загрузка вложений
- отображение статуса отправки в модальном окне

Поддерживаемые форматы файлов:

- `jpg`
- `jpeg`
- `png`
- `pdf`
- `txt`
- `doc`
- `docx`
- `xls`
- `xlsx`

Максимальный размер одного файла:

- `10 MB`

---

## Менеджерская панель

Маршруты:
- `/manager/dashboard`
- `/manager/tickets`
- `/manager/tickets/{ticketId}`

Возможности:

- список тикетов с пагинацией
- фильтр по email
- фильтр по телефону
- фильтр по статусу
- быстрый фильтр по периоду: день / неделя / месяц
- просмотр статистики по статусам
- просмотр детальной карточки тикета
- скачивание вложений

---

## Статусы тикетов

В проекте используются 3 статуса:

- `New` — новая заявка
- `InProgress` — в обработке
- `Processed` — обработана

## Роли и права

Используется пакет `spatie/laravel-permission`.

- `manager`

`manager` получает доступ к:

- просмотру клиентов
- просмотру тикетов
- изменению статуса тикетов

## Локальный запуск

### 1. Клонировать проект

```bash
git clone <repo-url>
cd mini-iframe-crm
```

### 2. Установить PHP-зависимости

```bash
composer install
```

### 3. Создать `.env`

```bash
cp .env.example .env
```

### 4. Сгенерировать ключ приложения

```bash
php artisan key:generate
```

### 5. Создать SQLite-файл

```bash
touch database/database.sqlite
```

Для Windows можно создать пустой файл вручную в папке `database`.

### 6. Выполнить миграции и сиды

```bash
php artisan migrate:fresh --seed
```

### 7. Установить frontend-зависимости

```bash
bun install
```

### 8. Запустить проект

```bash
composer run dev
```

После запуска будут подняты:

- Laravel server
- очередь
- Vite dev server

---

## Запуск в Docker

### 1. Перейти в корень проекта

```bash
cd mini-iframe-crm
```

### 2. Собрать и запустить контейнер

```bash
docker compose up --build
```

Контейнер автоматически:

- создаст `database/database.sqlite`
- подготовит `.env`
- установит `composer`-зависимости
- установит `bun`-зависимости
- сгенерирует `APP_KEY`
- выполнит миграции и сиды
- запустит Laravel, очередь и Vite

### 3. Открыть проект

- приложение: `http://127.0.0.1:8000`
- Vite dev server: `http://127.0.0.1:5173`

### Другой способ запуска

Запуск в фоне:

```bash
docker compose up --build -d
```

Остановить контейнеры:

```bash
docker compose down
```

Посмотреть логи:

```bash
docker compose logs -f
```

## Как проверить функционал

### 1. Отправка заявки через виджет

1. Открой `http://127.0.0.1:8000/widget`
2. Заполни форму обратной связи
3. При необходимости прикрепи файлы
4. Нажми кнопку отправки
5. Убедись, что заявка успешно отправлена

### 2. Проверка тикета в менеджерской панели

1. Открой `http://127.0.0.1:8000/manager/dashboard`
2. Перейди в раздел тикетов
3. Найди созданную заявку в списке
4. Открой карточку тикета
5. Проверь данные клиента, текст обращения и вложения

### 3. Проверка фильтров и статусов

1. На странице тикетов отфильтруй обращения по email, телефону или статусу
2. Используй фильтр по периоду: день / неделя / месяц

### 4. Проверка статистики

1. Открой страницу `manager/tickets`
2. Посмотри карточки статистики
3. Проверь количество тикетов по статусам за:
   день
   неделю
   месяц
